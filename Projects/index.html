Personal Project

Penalty Shootout Game

A mobile penalty kick game built in MIT App Inventor. Players flick a soccer ball toward the goal while a goalkeeper moves to block shots. The app tracks your score as you try to beat the keeper with well-placed strikes. Built as a personal project to explore game mechanics, sprite animation, and touch-based interaction on mobile.

MIT App Inventor Mobile Game Design
Class Project

Top Choices Survey App

An interactive survey application that lets users vote for their favorite ice cream flavor or submit a new one. Built using best-so-far loop logic to dynamically determine the winning flavor and display real-time vote tallies. Features include custom flavor submission, spell correction, and a live leaderboard of results.
