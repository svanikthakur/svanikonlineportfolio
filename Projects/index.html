Personal Project

Penalty Shootout Game

This is a project I created from scratch without steps. In this project, I was asked to build a project from scratch about something that I am passionate about. I chose soccer because it is one of my favorite sports. In this app, you have to try and shoot a soccer ball into a goal, as you reach certain checkpoints for goals scored, the levels get harder and the difficulty increases. This game proved to be very fun and I enjoyed building this app.

Class Project

Top Choices Survey App

This project was an extremely fun project to code as it exposed me to the beginning of loops and the best-so-far loop. In this project, you survey multiple people on their favorite flavor of ice cream, they can choose an existing flavor, or they can add their own which will then be added to the poll. It turned out to be a very successful project and I look forward to using best-so-far loops again in the future.
